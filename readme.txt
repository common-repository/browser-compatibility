Browser compatible provides the way to manage their theme at supported browsers. 

== Description ==

It provides a very easy interface to select your respective browsers and resolution for which your blog is fully functional otherwise audience or blog visitors will get a custom warning message as set by the blog administrator. Blog admin have the option to set the warning message as per the theme by choosing the colour picker.  

== Installation ==

Upload the Browser Compatible plugin to your blog and Activate it. Under the setting menu there is sub menu name as 'Browser compatibility'. here you can set your option accordingly and enjoy.


1, 2, 3: You're done!
